# Contributing

Contributions are welcome. Please keep changes focused and easy to review.

Basic workflow:

1. Fork this repository.
2. Create a branch for your change.
3. Modify code or documentation.
4. Build locally:

   ```powershell
   .\gradlew.bat --no-daemon plugin
   ```

5. Submit a Pull Request.
6. In the PR description, explain the purpose of the change and how it was tested.

Please avoid unrelated refactors in bug fixes or small documentation updates.
